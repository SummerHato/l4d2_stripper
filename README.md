这里是一些自己的战役stripper改图文件,大部分只是开局放枪

用法:将文件夹放到服务器addons里

bt4.cfg:三方图lockdown的m4,同上,解决了真人玩家佐伊会被传送卡死的问题

c6m1_riverbank.cfg:官图短暂时刻m1,开局放置小枪,删除一代人物生成及相关实体

c6m3_port.cfg:官图短暂时刻m3,删除一代人物生成及相关实体
